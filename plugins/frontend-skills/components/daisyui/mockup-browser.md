# Browser mockup

Browser mockup shows a box that looks like a browser window.

## Class Reference

| Class | Description | Category |
|-------|-------------|----------|
| `mockup-browser` | Browser mockup | component |
| `mockup-browser-toolbar` | Toolbar part which can include addressbar, etc | part |

## Examples

### browser mockup with border
<div class="mockup-browser border border-base-300 w-full">
  <div class="mockup-browser-toolbar">
    <div class="input">https://daisyui.com</div>
  </div>
  <div class="grid place-content-center border-t border-base-300 h-80">Hello!</div>
</div>
