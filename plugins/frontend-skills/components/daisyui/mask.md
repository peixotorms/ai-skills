# Mask

Mask crops the content of the element to common shapes.

## Class Reference

| Class | Description | Category |
|-------|-------------|----------|
| `mask` | Masks the content with shape | component |
| `mask-squircle` | squircle | style |
| `mask-heart` | heart | style |
| `mask-hexagon` | hexagon vertical | style |
| `mask-hexagon-2` | hexagon horizontal | style |
| `mask-decagon` | decagon | style |
| `mask-pentagon` | pentagon | style |
| `mask-diamond` | diamond | style |
| `mask-square` | square | style |
| `mask-circle` | circle | style |
| `mask-star` | star | style |
| `mask-star-2` | star (bold) | style |
| `mask-triangle` | triangle pointing top | style |
| `mask-triangle-2` | triangle pointing down | style |
| `mask-triangle-3` | triangle pointing left | style |
| `mask-triangle-4` | triangle pointing right | style |
| `mask-half-1` | Crops only the first half of mask | modifier |
| `mask-half-2` | Crops only the second half of mask | modifier |
