# Link

Link adds the missing underline style to links.

## Class Reference

| Class | Description | Category |
|-------|-------------|----------|
| `link` | Adds underline | component |
| `link-hover` | Only shows underline on hover | style |
| `link-neutral` | neutral color | color |
| `link-primary` | primary color | color |
| `link-secondary` | secondary color | color |
| `link-accent` | accent color | color |
| `link-success` | success color | color |
| `link-info` | info color | color |
| `link-warning` | warning color | color |
| `link-error` | error color | color |

## Examples

### Link
<button class="link">Click me</button>

```html
<a class="link">Click me</a>
```
