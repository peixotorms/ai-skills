# Progress

Progress bar can be used to show the progress of a task or to show the passing of time.

## Class Reference

| Class | Description | Category |
|-------|-------------|----------|
| `progress` | For <progress> tag | component |
| `progress-neutral` | neutral color | color |
| `progress-primary` | primary color | color |
| `progress-secondary` | secondary color | color |
| `progress-accent` | accent color | color |
| `progress-info` | info color | color |
| `progress-success` | success color | color |
| `progress-warning` | warning color | color |
| `progress-error` | error color | color |
