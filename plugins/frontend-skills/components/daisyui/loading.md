# Loading

Loading shows an animation to indicate that something is loading.

## Class Reference

| Class | Description | Category |
|-------|-------------|----------|
| `loading` | Loading element | component |
| `loading-spinner` | spinner animation | style |
| `loading-dots` | dots animation | style |
| `loading-ring` | ring animation | style |
| `loading-ball` | ball animation | style |
| `loading-bars` | bars animation | style |
| `loading-infinity` | infinity animation | style |
| `loading-xs` | Extra small size | size |
| `loading-sm` | Small size | size |
| `loading-md` | Medium size | size |
| `loading-lg` | Large size | size |
| `loading-xl` | Extra large size | size |
