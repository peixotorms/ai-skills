# Frontend Skills

Frontend development guidelines for Claude Code — currently focused on **Tailwind CSS** with plans to expand to other frameworks.

## Skills

### Tailwind CSS (6 skills)

| Skill | Activates When |
|-------|---------------|
| `tailwind-guidelines` | Writing, reviewing, or refactoring any Tailwind CSS code |
| `tailwind-components` | Building UI components (buttons, forms, modals, tables, etc.) |
| `tailwind-layouts` | Building page layouts, app shells, navigation structures |
| `tailwind-ecommerce` | Building ecommerce interfaces, product pages, carts, checkout |
| `tailwind-marketing` | Building marketing pages, landing pages, heroes, pricing |
| `tailwind-dashboard` | Building admin dashboards, data tables, CRUD interfaces |

### Sources

Guidelines derived from:
- [Tailwind CSS](https://tailwindcss.com/docs) official documentation (v4.x)
- [Headless UI](https://headlessui.com/) component patterns
- [Tailwind UI](https://tailwindui.com/) component design patterns
- [Flowbite](https://flowbite.com/) block and dashboard patterns

## Commands

| Command | Description |
|---------|-------------|
| `/tailwind-review [file]` | Review code against Tailwind CSS best practices |

## Installation

```bash
claude plugin install frontend-skills --source ./plugins/frontend-skills
```
